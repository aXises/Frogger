# Frogger
CSSE2010 AVR Project

## Specifications / Requirements
Baud Rate - 19200

ATMEGA324a Board

8 x 16 LED Matrix

Arduino Joystick

Piezo buzzer

## IO
|               | PIN7 | PIN6 | PIN5 | PIN4 | PIN3 | PIN2 | PIN1 | PIN0 |
| ------------- |------| -----|------|------|------|------|------|------|
| PORTA         | - | L/R | U/D | L4 | L3 | L2 | L1 | L0 |
| PORTB         | SCK | MISO | MOSI | SS | B3 | B2 | B1 | B0 |
| PORTC         | DP | G | F | E | D | C |B | A|
| PORTD         | - | - | S6 | BUZZER | S7 | CC | RX | TX |
